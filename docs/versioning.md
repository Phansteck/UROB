# UR0B Versioning Mandate (VR–001)
Unified Refining Oversight Bureau – Official Development Policy

## 1. Overview
This document defines the version numbering rules used throughout development of the Unified Refining Oversight Bureau (UR0B) mod for Factorio.

## 2. Version Format
UR0B uses:
MAJOR.MINOR.PATCH

## 3. PATCH Versions (0.0.x)
Used for non-playable development. May increase indefinitely.

## 4. MINOR Versions (0.1.x, 0.2.x)
Only permitted when:
- The mod loads without errors  
- A new game can be started with UR0B enabled  
- At least one feature functions in gameplay  

## 5. MAJOR Versions (1.x.x)
Reserved for feature-complete overhaul releases.

## 6. Summary
PATCH = scaffolding  
MINOR = playable  
MAJOR = complete

## 7. Enforcement
Only the project lead (Phansteck) may authorize deviations.
